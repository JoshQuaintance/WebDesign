Normal JavaScript DOM

//Without jQuery
var Playlist = document.getElementById("spotify");

function spotifyPlaylist() {
	if (Playlist.style.display === "none") {
	Playlist.style.display = "inline";
	} else {
	Playlist.style.display = "none";
	}
}
	
Javascript With jQuery

//Normal Javascript 
function spotifyPlaylist() {
	//if the playlist is displayed, then run this
	if ($("#spotify").css("display") === "none") {
		$("#spotify").css("display", "inline")
	} //otherwise run this one instead 
		else {
		$("#spotify").css("display", "none")
	}
}

Mistakes a lot of starter make (Even Me)

(Too Lazy To Read? Just Read The Red)

A lot of starters, always think that professional programmers sit on their computer all day typing, that the codes just come flying in. That is a wrong mindset, all you need to memorize for coding is probably the boilerplate and some basic stuff. Even all of those easy stuff have shortcuts that is integrated in some code editor. It's not a bad thing to forget codes, just Google them. That's what every programmers do, even the professional ones. They Google what they need, NOT MEMORIZE them.
